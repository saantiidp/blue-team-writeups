
# Blue Team Labs – Writeups

Este repositorio contiene una colección de writeups y notas técnicas basadas en laboratorios prácticos de ciberseguridad enfocados en defensa (Blue Team), principalmente de plataformas como CyberDefenders y Blue Team Labs.

El objetivo de este repositorio es documentar el proceso de análisis, las metodologías utilizadas y los aprendizajes obtenidos en escenarios de:
- Network Forensics
- Análisis de logs
- Detección de actividad maliciosa
- Incident Response
- Threat Hunting básico

## Estructura del repositorio

Cada archivo corresponde a un laboratorio distinto e incluye:
- Descripción del escenario
- Objetivos del análisis
- Herramientas utilizadas
- Metodología seguida
- Hallazgos principales
- Conclusiones y aprendizajes


## Enfoque

Los writeups están orientados a:
- Explicar el proceso de análisis, no solo el resultado
- Practicar documentación técnica clara y estructurada
- Reforzar conceptos de defensa y análisis forense
- Servir como portfolio personal de aprendizaje en Blue Team

No se incluyen soluciones oficiales ni respuestas directas a los retos, sino el razonamiento y las técnicas aplicadas durante el análisis.

## Herramientas habituales

Dependiendo del laboratorio, se utilizan herramientas como:
- Wireshark
- SIEM (Splunk / Elastic u otros)
- Análisis de logs en sistemas Windows y Linux
- Utilidades básicas de forense y análisis de red

## Nota

Este repositorio tiene fines educativos y de aprendizaje.  
Todo el contenido está basado en escenarios simulados de laboratorio.
